微信小程序学习记录

发表于 2017-08-07 | 分类于 记录 | | 阅读次数

公司要开发个小程序官网，所以就学习下微信小程序，顺带记录一下要点

项目配置

根目录

  • app.js - 入口文件
  • app.json - 配置文件
  • app.wxss - css主体

app.json

决定页面文件的路径、窗口表现、设置网络超时时间、设置多 tab 、是否开启debug模式

pages

整个小程序的路由配置

windows

用于设置小程序的状态栏、导航条、标题、窗口背景色

tabBar

底部tabBar设置

networkTimeout

网络请求的设置

debug

调试信息以info输出到控制面板

页面配置：

每个页面由4个文件组成：js、wxml、wxss、json。分别对应的是逻辑、结果、样式、配置

逻辑层

App()用来注册小程序，接受一个object参数

生命周期

App({
  onLaunch: function(options) {
    // 初始化，页面还没生成
  },
  onShow: function(options) {
      // 从后台进入前台显示
  },
  onHide: function() {
      // 从前台进入后台
  },
  onError: function(msg) {
    console.log(msg)
  },
  // 可以添加任意的数据进入参数里，然后通过this调用
  globalData: 'I am global data'
})

// 全局函数
getApp()

在onLaunch()和onShow的参数可以获取一些信息数据、场景值

页面注册

Page({
  data: {
    // 声明属性
    text: "This is page data."
  },
  // 注册页面的生命周期
  onLoad: function(options) {
    // Do some initialize when page load.
  },
  onReady: function() {
    // Do something when page ready.
  },
  onShow: function() {
    // Do something when page show.
  },
  onHide: function() {
    // Do something when page hide.
  },
  onUnload: function() {
    // Do something when page close.
  },
  onPullDownRefresh: function() {
    // Do something when pull down.
  },
  onReachBottom: function() {
    // Do something when page reach bottom.
  },
  onShareAppMessage: function () {
   // return custom share data when user share.
  },
  onPageScroll: function() {
    // Do something when page scroll
  },
  // 自定义事件
  viewTap: function() {
    this.setData({
      text: 'Set some data for updating view.'
    })
  },
  customData: {
    hi: 'MINA'
  }
})
